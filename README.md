Discription:
This project is a redesign of the original Space Jam (1996) website. The goal was to improve the user experience (UI/UX) while keeping similar content. 

Purpose: 
The original website is interactive but difficult to navigate and not user-friendly by modern standards. 
This redesign focuses on making the content clearer, more structured, and easier to use. 

Improvments: 
- Cleaner and more modern layout 
- Better navigation 
- Addad character images for a more visual experience
-inspired by modern streaming platforms (like Netflix)
-Responsive design for mobile devices

Changes to content: 
Some parts of the original website were removed because the made the page confusing and harder to use.
Instead, i focused on the most important content such as:
-Movie information
-Main characters

Technologies Used: 
HTML
SCSS
Node.js (for compiling SCSS)

Project Structure
src/scss/ - SCSS files (partials + main file)
dist/ - Compiled CSS
index.html - Main Page
